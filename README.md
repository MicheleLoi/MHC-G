# MHC-G — Meaningful Human Control, Germinal

The germinal cell: a minimal working habit for staying in control
when AI does real work on your files.

## What it is

MHC-G is a conversation pattern for Claude Cowork. It ensures that
when AI does consequential work on your files, you stay in control
of decisions — and have a readable record of what happened.

Five rules govern how Claude behaves: state intent before acting,
surface choices during, write up what happened after, make
persistence visible, and hold coherence across sessions.

## Who it's for

Someone using Claude Cowork for real work — writing, research,
analysis — who wants to stay oriented and in control without
learning a system.

## How to get it

Clone or download this repository. Select the MHC-G folder directly
in Claude Cowork. That's it — CLAUDE.md loads automatically and
the habit begins.
